# Jiakai Wang Personal Website

A modern, modular Jekyll-based personal website showcasing research themes, projects, and technical documentation.

## Quick Start

To run the site locally:
```bash
bundle exec jekyll serve -l -H localhost
```

## Key Features

- **Modular Architecture**: ES6 modules for JavaScript, component-based CSS
- **Data-Driven Content**: YAML data files for easy content management
- **Glass Morphism Design**: Modern glass effect UI components
- **Event Delegation**: Declarative event handling via data-action attributes
- **Responsive Design**: Mobile-first, works on all devices
